Teams Cache Exporter

Export the currently open Microsoft Teams web chat from browser cache into cleaned JSON, CSV, and Markdown files.

Privacy Policy 隐私政策 GitHub Repository

This site is intended for GitHub Pages hosting and browser store review links.

What This Extension Does

  • Exports only the Teams chat that is currently open in the browser.
  • Reads locally cached Teams data from browser IndexedDB.
  • Downloads cleaned JSON, CSV, and Markdown transcript files.
  • Optionally includes the raw cache dump and can hide system messages.
