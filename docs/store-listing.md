# Store Listing Copy

## Chrome Web Store Short Description

Export the current Microsoft Teams web chat from local browser cache into cleaned JSON, CSV, and Markdown.

## Chrome Web Store Long Description

Teams Cache Exporter helps you export the currently open Microsoft Teams web chat from your browser cache.

It is designed for Teams Web v2 when official self-service export options are unavailable, incomplete, or restricted by account type.

What it does:

- exports only the chat that is currently open
- reads locally cached Teams data from the browser
- generates cleaned JSON, CSV, and Markdown transcript files
- can optionally include the raw cache dump for debugging
- can optionally hide system messages from the final export

What it does not do:

- no third-party backend
- no chat upload to our servers
- no ads
- no analytics

Important notes:

- exported data is limited to what Teams has already cached locally in the current browser profile
- this is not an official Microsoft export path
- users are responsible for complying with organizational and school policies before exporting Teams content
