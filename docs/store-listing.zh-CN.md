# 商店上架文案

## Chrome Web Store 短描述

从浏览器本地缓存导出当前打开的 Microsoft Teams 网页聊天，并生成清洗后的 JSON、CSV 和 Markdown 文件。

## Chrome Web Store 长描述

Teams Cache Exporter 用来导出当前正在浏览器里打开的 Microsoft Teams 聊天。

它主要面向 Teams Web v2，适合官方自助导出不可用、不完整，或者受账号类型限制的场景。

主要功能：

- 只导出当前打开的聊天
- 读取浏览器本地缓存中的 Teams 数据
- 生成清洗后的 JSON、CSV 和 Markdown 纪要
- 可选附带原始缓存导出，便于调试
- 可选隐藏系统消息

不会做的事情：

- 没有第三方后端
- 不会把聊天上传到我们的服务器
- 没有广告
- 没有分析埋点

重要说明：

- 能导出的内容受限于当前浏览器配置文件里已经缓存下来的 Teams 数据
- 这不是微软官方导出路径
- 用户需要在导出前自行确认符合学校、公司或组织的相关政策
